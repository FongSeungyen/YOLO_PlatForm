## 3. 数据集管理操作

### 3.1 创建新数据集

**步骤：**
1. 点击侧边栏 **"📁 数据集"**
2. 选择任务类型（Classify/Detect/Segment/Pose/OBB）
3. 点击 **"新建数据集"** 按钮
4. 填写数据集信息：
   - 数据集名称（必填）
   - 数据集描述（可选）
   - 保存位置（默认 `workspace/datasets/`）
5. 配置类别信息：
   - 添加类别名称（每行一个）
   - 选择类别颜色
   - 姿态估计任务需配置关键点
6. 点击 **"创建"** 完成

### 3.2 导入现有数据

**支持的方式：**
- 导入文件夹（拖拽或选择）
- 导入压缩包（ZIP）
- 导入特定格式（COCO/YOLO/VOC等）

**导入步骤：**
1. 点击 **"导入数据"**
2. 选择导入方式
3. 选择数据源
4. 配置数据分割比例（训练/验证/测试）
5. 点击 **"开始导入"**

### 3.3 数据集操作

| 操作 | 说明 |
|------|------|
| **打开** | 进入数据集详情页 |
| **标注** | 打开标注界面 |
| **导出** | 导出为其他格式 |
| **复制** | 复制数据集 |
| **删除** | 删除数据集（确认后执行） |

## 4. 数据标注操作

### 4.1 打开标注界面

1. 在数据集列表中选择数据集
2. 点击 **"标注"** 按钮
3. 选择要标注的图像

### 4.2 标注工具使用

#### 4.2.1 目标检测标注

**绘制边界框：**
1. 选择 **矩形工具**（快捷键：R）
2. 在图像上拖拽绘制矩形框
3. 选择类别标签
4. 按 **Enter** 确认

**调整边界框：**
- **移动**：拖拽框中心
- **调整大小**：拖拽边角手柄
- **删除**：选中后按 **Delete**

#### 4.2.2 实例分割标注

**绘制多边形：**
1. 选择 **多边形工具**（快捷键：P）
2. 点击图像创建顶点
3. 双击或按 **Enter** 闭合多边形
4. 选择类别标签

**编辑多边形：**
- **添加顶点**：在边上点击
- **删除顶点**：选中顶点后按 **Delete**
- **移动顶点**：拖拽顶点

#### 4.2.3 姿态估计标注

**标注关键点：**
1. 选择 **关键点工具**（快捷键：K）
2. 点击图像放置关键点
3. 选择关键点名称
4. 设置可见性（可见/遮挡/未标注）

**快捷键：**
| 快捷键 | 功能 |
|--------|------|
| **1-9** | 快速选择第N个关键点 |
| **V** | 切换可见性 |
| **Tab** | 下一个图像 |

#### 4.2.4 OBB标注

**绘制旋转框：**
1. 选择 **旋转框工具**（快捷键：O）
2. 拖拽绘制矩形
3. 拖拽旋转手柄调整角度
4. 选择类别标签

**精确调整：**
| 快捷键 | 功能 |
|--------|------|
| **Q/E** | 旋转框（-/+1度） |
| **Shift+Q/E** | 旋转框（-/+5度） |

### 4.3 AI辅助标注

**使用步骤：**
1. 点击 **"🤖 AI标注"** 按钮
2. 选择预训练模型
3. 配置推理参数：
   - 置信度阈值（默认0.25）
   - IoU阈值（默认0.45）
4. 点击 **"开始标注"**
5. 审核并修正AI生成的标注
6. 点击 **"保存标注"**

## 5. 模型训练操作

### 5.1 创建训练任务

1. 点击侧边栏 **"🚀 训练"**
2. 点击 **"新建训练"**
3. 选择数据集
4. 配置训练参数（见下表）
5. 点击 **"开始训练"**

### 5.2 训练参数配置

| 参数 | 默认值 | 说明 |
|------|--------|------|
| **模型** | yolo11n | 选择YOLO模型架构 |
| **Epochs** | 100 | 训练轮数 |
| **Batch Size** | 16 | 批次大小 |
| **Image Size** | 640 | 输入图像尺寸 |
| **Learning Rate** | 0.01 | 初始学习率 |
| **Optimizer** | SGD | 优化器（SGD/Adam/AdamW） |
| **Device** | auto | 训练设备（auto/cpu/cuda） |

### 5.3 训练监控

**实时监控：**
- 损失曲线图
- mAP变化曲线
- 学习率曲线
- 当前Epoch进度
- GPU显存使用

**操作按钮：**
| 按钮 | 说明 |
|------|------|
| **⏸️ 暂停** | 暂停训练（可恢复） |
| **⏹️ 停止** | 停止训练 |
| **📊 日志** | 查看详细训练日志 |
| **💾 保存** | 保存当前最佳模型 |

### 5.4 断点续训

**自动续训：**
- 系统会自动保存训练检查点
- 重启后自动检测未完成的训练
- 点击 **"继续训练"** 恢复

**手动续训：**
1. 选择之前的训练任务
2. 点击 **"加载检查点"**
3. 选择要恢复的检查点
4. 点击 **"继续训练"**

## 6. 模型评估操作

### 6.1 运行评估

1. 点击侧边栏 **"📊 评估"**
2. 选择训练好的模型
3. 选择评估数据集
4. 点击 **"开始评估"**

### 6.2 查看评估结果

**评估报告包含：**
- **性能指标**：mAP、Precision、Recall、F1
- **混淆矩阵**：各类别预测分布
- **PR曲线**：Precision-Recall曲线
- **样例预测**：可视化预测结果

### 6.3 模型对比

1. 在评估界面点击 **"对比"**
2. 选择多个模型
3. 选择对比维度
4. 点击 **"生成对比报告"**

## 7. 模型部署操作

### 7.1 本地推理

1. 点击侧边栏 **"🌐 部署"**
2. 选择 **"本地推理"**
3. 选择模型
4. 上传图像或选择图像文件夹
5. 查看推理结果

### 7.2 导出模型

**导出步骤：**
1. 选择 **"导出模型"**
2. 选择模型
3. 选择导出格式（ONNX/TensorRT/TFLite等）
4. 配置导出参数：
   - 输入尺寸
   - 批次大小
   - 动态轴（ONNX）
5. 点击 **"导出"**

**支持格式：**
| 格式 | 适用场景 |
|------|----------|
| **ONNX** | 跨平台部署 |
| **TensorRT** | NVIDIA GPU加速 |
| **TFLite** | 移动端/嵌入式 |
| **CoreML** | Apple设备 |
| **OpenVINO** | Intel设备 |

### 7.3 启动API服务

1. 选择 **"API服务"**
2. 选择模型
3. 配置服务参数：
   - 端口号（默认8000）
   - 工作进程数
   - 请求超时时间
4. 点击 **"启动服务"**
5. 查看API文档和测试接口

**API端点：**
- `POST /predict` - 单图像推理
- `POST /predict/batch` - 批量推理
- `GET /health` - 健康检查
- `GET /info` - 模型信息

## 8. 系统工具使用

### 8.1 系统诊断

**功能：** 检查系统环境和配置

**使用方法：**
1. 点击菜单 **"工具 → 系统诊断"**
2. 点击 **"开始诊断"**
3. 查看诊断结果

**诊断项：**
- Python版本
- PyTorch版本
- CUDA/cuDNN版本
- GPU信息
- 磁盘空间
- 内存使用

### 8.2 缓存清理

**功能：** 清理系统缓存文件

**使用方法：**
1. 点击菜单 **"工具 → 缓存清理"**
2. 选择要清理的项目
3. 点击 **"开始扫描"**
4. 查看可清理的文件
5. 点击 **"清理"** 执行

**可清理项目：**
- Python缓存（__pycache__）
- 临时文件
- 日志文件
- 图像缩略图缓存

### 8.3 日志查看器

**功能：** 查看应用日志

**使用方法：**
1. 点击菜单 **"工具 → 日志查看器"**
2. 选择日志级别过滤
3. 使用搜索框查找关键词
4. 点击 **"导出"** 保存日志

## 11. 常见问题操作

### 11.1 界面显示异常

**问题**：界面显示不完整或错位

**解决**：
1. 尝试切换主题（深色/浅色）
2. 调整窗口大小
3. 在设置中调整字体大小
4. 更新显卡驱动

### 11.2 模型下载失败

**问题**：首次使用AI标注时模型下载失败

**解决**：
1. 检查网络连接
2. 配置代理设置（如需）
3. 手动下载模型放到 `workspace/models/` 目录

### 11.3 训练显存不足

**问题**：训练时显存溢出

**解决**：
1. 减小批次大小（Batch Size）
2. 减小输入图像尺寸（Image Size）
3. 使用更小的模型（如 yolo11n 替代 yolo11x）
4. 启用混合精度训练

### 11.4 标注保存失败

**问题**：标注无法保存

**解决**：
1. 检查磁盘空间
2. 检查目录写入权限
3. 确保标注格式正确
4. 查看日志获取详细错误信息

## 12. 最佳实践

### 12.1 数据集准备

1. **图像质量**：使用清晰、光线充足的图像
2. **数据量**：每个类别至少100张图像
3. **数据多样性**：包含不同角度、光照、背景
4. **标注质量**：确保标注准确、一致

### 12.2 模型训练

1. **数据增强**：适当使用数据增强提高泛化能力
2. **学习率**：根据数据集大小调整学习率
3. **早停**：使用早停避免过拟合
4. **验证**：定期在验证集上评估模型

### 12.3 模型部署

1. **模型优化**：导出前进行量化和剪枝
2. **性能测试**：在目标设备上测试推理速度
3. **错误处理**：实现完善的错误处理和日志记录
4. **监控**：部署后持续监控模型性能
